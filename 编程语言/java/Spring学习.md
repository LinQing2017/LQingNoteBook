# Nacos应用场景

- 服务的注册与发现：实现应用进行RestAPI调用时的负载均衡，应用可以不需要知道服务端的确切地址（使用服务名称即可），从而实现配置管理。

- 配置中心：

    - 提供一个配置中心的功能
    - 并且实现一些配置的动态刷新
    - 多种方式的配置文件管理
    - 多文件加载以及共享配置

- dobbo整合
